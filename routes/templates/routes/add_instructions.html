Add Route

Instructions

Important Info: You will need to carry out the following instructions on the Google Maps desktop version as the code you will receive is not available on the mobile version of Google Maps.

Step 1: When you have created your route in Google Maps using the bicycle tab, click on the menu button in the top left of the screen.

Step 2: From the menu select 'Share or embed map'.

Step 3: On the next screen select the tab 'Embed a map'.

Step 4: On the final screen select the 'COPY HTML' link. This will copy the map code on to your clipboard.

Step 5: Finally paste the map code into the 'Google Map Code' field of the Add Route form.
